fetch_data.py

Description:
- Fetches current-season data from the football-data.org API for the top five European leagues.
- Collects teams, competitions, standings, matches, and scores and normalizes them into CSV tables.
- Produces CSV outputs that mirror the repository's dataset schema (leagues, teams, stadiums, matches, scores, standings).
